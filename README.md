# dailynotes

## Jonathans daily notes

See my [Daily Notes](dailylogs) for links and stuff I log that I have found interesting

Ok, this is for my daily notes and links. Some of you may find interesting to see what I look at when I have time and inclination to jot down. Most are links in my dailylogs/ folder.

I seem to have a plethora of places to store notes and links, so why not try GitHub - I have tried:

* Obsidian - https://obsidian.md/
* SimpleNote - https://app.simplenote.com/
* InfoSelect (my favourite but paid for)
* TiddlyWiki (brilliant personal wiki - https://tiddlywiki.com/)
* DokuWiki (good online resource and easy to manage / setup - https://www.dokuwiki.org/dokuwiki)
* RememberTheMilk (more a todo manager but input syntax is a keyboard warriors dream - https://www.rememberthemilk.com/)
* GitHub :)

Stuff that hits the top of the "this is interesting to me" at the moment:

* Glance (great dashboard for RSS, notes, links, feeds, shortcuts and a lot more - easy install and creates its own local webserver) https://github.com/glanceapp/glance
