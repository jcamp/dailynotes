## Chrome shortcut keys (debugger / essential)

The primary keyboard shortcut to open Chrome Developer Tools is **F12** or **Ctrl+Shift+I** (Windows/Linux) and **Cmd+Option+I** (Mac). To immediately open the Command Menu within DevTools—allowing you to run commands, files, or snippets—use **Ctrl+Shift+P** (Windows/Linux) or **Cmd+Shift+P** (Mac). [](https://developer.chrome.com/docs/devtools/shortcuts)

Essential Chrome DevTools Shortcuts 

- **Toggle Inspect Mode:** `Ctrl+Shift+C` (Windows) / `Cmd+Shift+C` (Mac).
- **Open Console Panel:** `Ctrl+Shift+J` (Windows) / `Cmd+Option+J` (Mac).
- **Search/Command Menu:** `Ctrl+Shift+P` (Windows) / `Cmd+Shift+P` (Mac).
- **Toggle Device Toolbar:** `Ctrl+Shift+M` (Windows) / `Cmd+Shift+M` (Mac).
- **Open File/Search:** `Ctrl+P` (Windows) / `Cmd+P` (Mac). [](https://developer.chrome.com/docs/devtools/shortcuts) 

General Chrome Browser Shortcuts 

- **Clear Browsing Data:** `Ctrl+Shift+Delete` (Windows) / `Cmd+Shift+Delete` (Mac).
- **Open New Tab:** `Ctrl+T` (Windows) / `Cmd+T` (Mac).
- **Reopen Last Closed Tab:** `Ctrl+Shift+T` (Windows) / `Cmd+Shift+T` (Mac).
- **View Source:** `Ctrl+U` (Windows) / `Cmd+Option+U` (Mac). [](https://www.dca.ca.gov/about_us/kbs_chrome.shtml)
    
For more, you can check the full [Chrome DevTools shortcuts](https://developer.chrome.com/docs/devtools/shortcuts) documentation.
