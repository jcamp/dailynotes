| Lubricant        | Main use               | Typical watch parts                             | Thickness     |
| ---------------- | ---------------------- | ----------------------------------------------- | ------------- |
| [Moebius 9010]() | High-speed pivots      | Balance jewels, escape wheel, fourth wheel      | Thin          |
| [Moebius 9104]() | Higher-load bearings   | Centre wheel, barrel arbor, slower train pivots | Medium/heavy  |
| [Moebius 9415]() | Escapement lubrication | Pallet stones and impulse surfaces              | Sticky grease |

The important distinction is:

* **9010** = speed
* **9104** = load/pressure
* **9415** = escapement contact surfaces

So in practical terms:

* If a pivot spins very fast with tiny load → use 9010
* If a pivot turns slower under pressure → use 9104
* If two escapement surfaces lock and unlock → use 9415

That combination covers most vintage hand-wind watches very well. ([HS Walsh & Sons][1])

A lot of hobby restorers stop there initially because once you go deeper you start adding:

* 8200 for mainsprings
* 8217 for automatic slipping bridle
* DX or 9501 for keyless works
* 941 or 9415 variants for specialised escapements

For the sort of vintage mechanical watches you are likely restoring, those three are a very sensible place to start. ([Etsy][2])

The confusion comes because older watchmaking guides often recommended:

* 9010
* D5
* 9415

Modern hobby watchmakers often replace D5 with 9104/HP1300 because it is synthetic and more stable long-term. ([Reddit][1])

Molykote DX is still used, but for a different purpose:

* keyless works
* sliding pinions
* setting levers
* winding parts

Basically:

> If it rotates → oil
> If it slides → grease

That rule appears repeatedly in watch repair discussions. ([Watch Repair Talk][2])

So if you are buying only three lubricants:

* 9010
* 9104
* 9415

…you can service most simple vintage hand-wind watches reasonably well.
